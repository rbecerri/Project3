What is Malware?

Malware is software designed with the intent to inflict damage to a computer, server, client, or computer network after being introduced in some way into a target's computer. The code is described as computer viruses, worms, Trojan horses, ransomware, spyware, adware, and scareware, among other terms.

The Process Take a closer look.

We tackled a Kaggle challenge to predict a Windows machineâ€™s probability of getting infected by various families of malware, based on different properties of that machine. The telemetry data containing these properties and the machine infections were generated by combining heartbeat and threat reports collected by Microsoft's endpoint protection solution, Windows Defender.

Here is what we did:

Clean-up. Wading through data.

We imported over 8 million records with approximately 80 features.
We identified this as a classification problem. Nearly all were category variables but several quantitative columns were dropped. We then removed any variables with too many n/a values.

Random Forests. Happy little [decision] trees.

From here we used one hot encoding to vectorize the data so that we could perform the training. This involved the creation of over 9000 features. In order to trim down these variables we used a random Random Forest model to perform recursive feature engineering. This created a series of decision trees into which a random assortment of the features were entered. Once this was done we received an accuracy value of around 50 percent and a ranking of importance of each of these individual features.

Optimization. Speeding things up.

To speed up and optimize the model we grabbed any feature with an importance of greater than 1 percent. We then ran these again over the entirety of the data. This time we got an accuracy of 49 percent, repeating previous results.
